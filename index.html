Introducing
5th release

Turn.js is a set of widgets, events, effects and themes for catalogs and ebooks built on top of jQuery.

Some features that will make you love turn.js 5th release:

  • The page turning widget supports two effects, page curls and solid covers.
  • You can embed almost any type of content from texts to videos or links.
  • It comes with a responsive user interface that is mobile compatible and scale well with complex contents.

  • All the widgets recycle DOM elements for efficient use of the client's resources. So, a 100K pages flipbook will have the same performance as one with only 10.
  • Average 60 fps animations that feel smooth.
  • You can customize many options and modify the entire design of all samples.
  • API designed for simplicity.
  • Comes with many samples, including a pre-built web app in Backbone.js and a lightbox for websites.
  • Devices supported: iPad, iPhone, Kindle Fire, Samsung Galaxy Tab, Nexus Tab, Surface.
  • Web browsers: Chrome, Firefox, Safari, IE 10
